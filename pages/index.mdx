# Welcome to the CF Identity Wallet Alpha Program Documentation

Welcome to the comprehensive documentation for the Alpha program of Identity Wallet. This guide is designed to help you navigate through the various features and functionalities of Identity Wallet, ensuring a smooth and efficient experience.

The documentation is divided into two main parts:

## Tutorials

This section contains step-by-step guides to help you with the initial setup and use of Identity Wallet to vote in Cardano Ballot 2024. You will find instructions for app installation on both iOS and Android, onboarding, creating your first identifier, connecting with Cardano Ballot, verifying your wallet, logging in, and creating and signing ballots.

For detailed guidance, visit the [App Installation page](ballot/install.mdx) to get started.

## Want to learn more?

This section covers advanced aspects of Identity Wallet, including technical details, security features, and additional tutorials on advanced actions. Topics include KERI Multisig AID, connections management, account recovery, the credentials testing tool, passcode changes, and managing the operations password.

Explore further details in the [Want to learn more page](more).

We hope this documentation serves as a valuable resource in your journey with Identity Wallet. For any further assistance or feedback, please refer to the support section or contact our team directly.
